# License Plate Recognition Web App

Web application for detecting and validating Portuguese license plates using OCR and Flask.

# Overview
This project is a web-based application that uses Optical Character Recognition (OCR) to detect and validate Portuguese vehicle license plates. It integrates a Python backend with a user-friendly frontend interface.

# Technologies Used
- Python
- Flask
- PaddleOCR
- HTML, CSS, JavaScript

# Features
- License plate detection using OCR
- Validation of Portuguese license plate formats
- Web interface for user interaction
- Camera/image-based input
